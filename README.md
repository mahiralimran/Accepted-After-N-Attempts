# Competitive Programming Archive

### *a.k.a. "How many times can one person get WA?"*

A collection of solutions that went through:
confusion → denial → brute force → panic → acceptance.



## 📁 Structure

Problems are organized by platform and difficulty (because pretending to be organized feels productive).



## 🧩 Platforms Covered

* Codeforces
* LeetCode
* AtCoder
* Random problems that ruined my day


## 🧩 Philosophy

> “Any problem can be solved…
> after enough incorrect submissions.”
